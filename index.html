트러블슈팅

Oculus Link 연결 문제 +

문제:

Oculus Quest 2와 PC 간 연결이 불안정하여 VR 개발 테스트가 어려웠음

원인:

Oculus 설정 디렉토리의 캐시 파일 충돌

해결:

설정 디렉토리 삭제 후 재설정으로 연결 안정화

교훈:

하드웨어 연결 문제는 소프트웨어 캐시 초기화로 해결될 수 있음

Unity 충돌 감지 오류 +

문제:

게임 오브젝트 간 충돌 감지가 정상적으로 작동하지 않음

원인:

Rigidbody 컴포넌트의 잘못된 설정

해결:

Rigidbody의 Is Kinematic 설정 조정 및 Collider 설정 재검토

교훈:

Unity 물리 시스템의 기본 설정을 정확히 이해하는 것이 중요

입력 UI 오류 +

문제:

사용자 입력 UI에서 텍스트 처리 오류 발생

원인:

TextMeshPro 컴포넌트의 비동기 처리 문제

해결:

TextMeshPro 처리 로직 변경 및 코루틴을 활용한 비동기 처리 개선

교훈:

UI 컴포넌트의 생명주기를 고려한 프로그래밍 필요

Unity 버전 문제 +

문제:

프로젝트 호환성 문제로 인한 빌드 오류

원인:

Unity 버전 간 API 변경사항

해결:

Unity 2022 LTS로 업그레이드하여 안정성 확보

교훈:

LTS 버전 사용의 중요성과 버전 관리의 필요성

Python: Space Escape

플랫폼: Meta Quest 2

개발 기간: 2025.03 ~ 2025.06

기술 스택: Unity 2022, XR Toolkit, C#

소개

외계인과 함께 Python 코딩으로 우주선을 운행해 탈출하는 학습형 VR 게임입니다. 사용자는 VR 환경에서 직접 코드를 작성하며 우주선을 조작하고, 다양한 퍼즐을 해결하여 우주에서 탈출하는 것이 목표입니다.

주요 기능

  • VR 환경에서의 키보드 입력 시스템
  • 외계인 캐릭터와의 상호작용
  • 단계별 게임 진행 시스템
  • Python 코드 실행 및 검증
  • 몰입감 있는 우주 환경 구현

심박수 어플리케이션

개발 기간: 2024.07 ~ 2024.08

플랫폼: Android Studio + Wear OS

기술 스택: Kotlin, Firebase, Android SDK

소개

물류 현장 근로자의 건강 관리를 위한 심박수 모니터링 애플리케이션입니다. CJ대한통운 미래기술 챌린지 참가작으로 개발되었습니다.

주요 기능

일반회원 모드:

  • 회원가입 및 로그인 시스템
  • 실시간 심박수 그래프 확인
  • 건강 이상 신고 이력 조회
  • 개인 건강 데이터 관리

관리자 모드:

  • 전체 회원 데이터 관리
  • 시스템 설정 기능
  • 마스킹된 개인정보 표시
  • 통계 및 리포트 생성

아이더룸

웹 기술을 활용한 프론트엔드 개발 팀 프로젝트입니다. 사용자 친화적인 인터페이스와 반응형 디자인을 구현하였습니다.

주요 기능

  • 반응형 웹 디자인
  • 사용자 인터랙션 구현
  • 모던 웹 표준 준수
  • 크로스 브라우저 호환성

FPS GAME

기술 스택: OpenGL, GLSL, C++

프로젝트 유형: 개인 프로젝트

소개

OpenGL과 GLSL을 활용하여 개발한 3D FPS 게임입니다. 저수준 그래픽스 프로그래밍을 통해 3D 렌더링 파이프라인을 직접 구현하였습니다.

주요 기능

  • 3D 모델 렌더링
  • 실시간 조명 시스템
  • 텍스처 매핑
  • 충돌 감지 시스템
  • 파티클 시스템
  • 사운드 시스템
